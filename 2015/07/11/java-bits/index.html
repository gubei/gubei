java-bits

java

java符号位

java所有的数值基础类型int,double,float,byte都是有符号位的。

java位展示

int i = xcafe ; // hexadecimal case
int j = 045 ;    // octal case
int l = 42 ;     // decimal case
int z = b1100110011 // binary case

java数值类型的位数

int 32位
byte 8位
double 64位
float 32位

java基础位操作

1. `|` or 操作
2. `&` and 操作
3. `^` Xor 操作
4. `~` Not 操作
5. `<<` 左移位，右边补零
6. `>>>` 逻辑移位，右边补零
7. `>>` 算法移位，右边补符号位
